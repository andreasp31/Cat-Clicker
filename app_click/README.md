# AppClick

Este proyecto ha sido desarrollado con **Angular CLI** y **Electron** para crear una aplicación de escritorio conectada a **MongoDB Atlas**.

---

## Servidor de Desarrollo

Para iniciar el servidor de desarrollo local, ejecuta:

```bash
ng serve

Una vez que el servidor esté funcionando, abrir el navegador en http://localhost:4200/.
```

## Consrtuir la aplicación

Para compilar el proyecto y generar los archivos de producción:

```bash
ng build
```
